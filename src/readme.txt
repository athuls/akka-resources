To run both chat and TODO applications: do the following.
To compile chat and todo respectively, run ./compileEverything.sh and ./compileEverythingTodo.sh.
There are many sample shell scripts used for basic testing.
To run the main ones, run all the 10-100 combinations and change the parameters accordingly. The first two parameters
of each line are the number of users and number of messages. Ignore the last two parameters for these particular commands
(ScalableCreateUsers and CausalScalableChat (similarly for other layers too).
